## nvim 

Run ```:PackerSync``` to install plugins.

Edit ```nvim/lua/plugins.lua``` to edit plugins, and run ```:PackerUpdate``` to make changes.

Edit ```M.colorscheme2dir ``` in ```/nvim/core/colorschemes.lua``` to change colour theme.
